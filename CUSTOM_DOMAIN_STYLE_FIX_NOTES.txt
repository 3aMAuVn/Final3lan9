Custom-domain styling fix for www.magicmathmaster.com.au

- shared CSS now loads with a page-relative path
- shared JS now loads with a page-relative path
- shared topbar logo and flag assets now load with page-relative paths
- compiled CSS is also inlined in head as a fallback, so the site keeps its styling even if the CSS file path is delayed after a custom-domain switch

Recommended after upload:

   www.magicmathmaster.com.au
